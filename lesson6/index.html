懲戒処分の指針について

平成12年3月31日 職職―68（人事院事務総長通知）

1 指針の目的と基本原則

この指針は、懲戒処分がより厳正かつ公正に行われるよう、任命権者が処分量定を決定する際の「標準例」を示したものです。
処分の決定にあたっては、非違行為の動機、態様、結果、故意・過失の度合い、職責、社会的影響、過去の処分歴などを総合的に考慮して判断されます。

一般服務関係

  • 欠勤: 正当な理由なく21日以上の欠勤 → 免職
  • 秘密漏洩: 職務上の秘密を故意に漏らし公務に支障 → 免職 または 停職

公金・公物関係

  • 横領: 公金・公物を横領 → 免職
  • 紛失・盗難: 重大な過失により紛失等 → 戒告

公務外の非行

  • 飲酒運転: 酒酔い運転等で人を死傷 → 免職
  • 痴漢・わいせつ: 痴漢行為等 → 免職 または 停職

監督責任

  • 隠ぺい・黙認: 部下の非違を知りつつ隠ぺい → 停職 または 減給
  • 指導監督不適正: 指導に適正を欠いた → 減給 または 戒告
重要： 民間企業と異なり、公務員は「全体の奉仕者」としての立場から、勤務時間外の私生活上の非行（飲酒運転や暴力行為など）であっても、信用失墜行為として厳しく処分の対象となります。

実務 危機管理コラム：不測の事態（逮捕・勾留）への備え

「無断欠勤」のリスクを回避するために

万が一、捜査機関による逮捕等で身体拘束を受け、物理的に出勤・連絡ができなくなった場合、「職場への連絡」が最優先事項となります。連絡がないまま放置すると「無断欠勤」として扱われ、懲戒処分の対象（または処分の加重）となるリスクがあります。

推奨される対策
  • 家族への連絡先共有:
    本人が電話できない場合に備え、家族が職場の管理職へ「現在は出勤できない状況にある」旨を即座に伝えられるよう、緊急連絡先を共有しておくこと。
  • 弁護人への依頼:
    弁護士に対し、速やかに職場へ連絡を入れるよう依頼すること。弁護士からの連絡であれば、状況の正確な伝達と法的な権利保護（年次休暇の事後振替の相談など）がスムーズに行える可能性があります。

法務 法的救済と適正手続（弁護士の視点）

行政処分としての限界と救済

上記の指針はあくまで任命権者が行う「行政内部の処分基準」であり、司法（裁判所）による最終的な判断ではありません。
懲戒処分も行政処分の一種であるため、処分内容が著しく妥当性を欠く場合（裁量権の逸脱・濫用）は、違法として取り消される可能性があります。

不服申し立ての権利

処分に不服がある場合、公務員には以下の権利が保障されています。

  • 審査請求: 処分説明書の交付から3ヶ月以内に、人事院（国家公務員）や公平委員会（地方公務員）に対して審査を申し立てる制度。
  • 取消訴訟: 審査請求の裁決を経た後（一部例外あり）、裁判所に対して処分の取り消しを求める訴訟。
